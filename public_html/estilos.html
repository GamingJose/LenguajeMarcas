Bordes, márgenes, ids y clases

Nombre Definición
Border Establece un borde al elemento. Ejemplo:

Este es un párrafo con un borde
padding Le da relleno al elemento. Si no se especifica nada el espacio es a izquierda, derecho, arriba, o abajo.Si se da demasiado padding, llega a pasar una palabra por línea
margin Se puede usar para dar un margen respecto a los elementos que están a los lados, también se usa para centrar elementos.Se pueden utilizar hasta valores negativos, para solaparlos.
id Es un identificador. Permite asignar un id a un elemento y llamarlo desde las hojas de estilos, en vez de asignar ese estilo a todos los elementos.Ejemplo : p id=estudiante En safari, cualquier id que se llame igual que una funcion en javaScript, revienta javaScript. Para utilizarlo en ccs es : #id{ color:red }
class Una clase se utiliza para especificar que algo pertenece a una clase. Ejemplo p id="aitor" class="dual1". Podemos tener 15 p's con la misma clase, pero ese con otro id diferente.Para utilizarlo en css es : .clase{ color :blue} p {color:green}
